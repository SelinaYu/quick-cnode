# quick-node
---
### Tips
开始之前，请安装 toolkit 工具和调试器，了解平台预览版，为创建项目和预览效果做好准备。如未安装相应的环境，可到参考[环境搭建][1]

### QuickStart
```
# Clone the repo
git clone git@github.com:SelinaYu/quick-node.git

# Move into the repo
cd quick-node

# Start the watcher
npm run server
```
